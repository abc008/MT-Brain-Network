# MT-Brain Network
This repository contains a Pytorch implementation of [A multi-task two-path deep learning system for predicting the invasiveness of craniopharyngioma]. 


### Requirements
- Python 3.7.0
- PyTorch-1.6.0
- CUDA Version 10.2
- CUDNN 7.6.5_0

### Installation
- Install Python 3.7.0
- pip install -r requirements.txt
